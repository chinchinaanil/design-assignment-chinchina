1110 Sequence Detector (Overlapping) 


This  implements a Mealy Finite State Machine (FSM) to detect the binary sequence 1110 using Behavioral Verilog HDL.
The design supports overlapping sequence detection, making it efficient for real-time pattern recognition.


 Features

Detects sequence: 1110
Mealy FSM (output depends on state + input)
Supports overlapping detection
Faster output response (output in same clock cycle)
Synthesizable Verilog code
Testbench included for simulation


It's a FSM type mealy machine overlapping Sequence Detector
