2-to-4 Decoder using Verilog HDL


Aim

To design and simulate a 2-to-4 Decoder using Verilog HDL in Vivado.


Theory

A Decoder is a combinational logic circuit that converts binary information from n input lines to a maximum of 2ⁿ unique output lines.
A 2-to-4 Decoder has:
2 input lines (A1, A0)
4 output lines (Y3, Y2, Y1, Y0)
For every input combination, only one output is HIGH.



Result

The 2-to-4 Decoder was successfully designed and simulated using Verilog HDL in Vivado, and the output waveform matched the truth table. :
