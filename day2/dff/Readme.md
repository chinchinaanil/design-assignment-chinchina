D Flip-Flop Using Verilog HDL


Aim

To design and simulate a D Flip-Flop using Verilog HDL and verify its functionality through behavioral simulation.
Objective
To understand the operation of a positive edge-triggered D Flip-Flop and implement it using behavioral modeling.


Theory

A D (Data) Flip-Flop is a sequential circuit used to store one bit of information. It has one data input (D), one clock input (CLK), and one output (Q).
The output changes only at the positive edge of the clock and takes the value present at the D input.Applications
Registers
Counters
Shift Registers
Memory Elements
Data Storage Circuits


Result

The D Flip-Flop was successfully designed and simulated using Verilog HDL. The output waveform verified the correct operation of the positive edge-triggered D Flip-Flop. :::
